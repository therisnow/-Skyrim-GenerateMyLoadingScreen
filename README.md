# Generate My Loading Screen

A Skyrim SE/AE custom loading screen generator.

## Downloads

Please download the latest packaged version from the **Releases** page.

## Languages

- [English Guide](README_EN.md)
- [中文说明](README_CN.md)

## Important

This is a generator tool, not the final in-game mod.  
Run the script first, then install the generated ZIP from the `output` folder with MO2 or Vortex.
